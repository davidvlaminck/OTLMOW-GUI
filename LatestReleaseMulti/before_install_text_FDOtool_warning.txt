CAUTION!
If you want to work with SDF files in OTL Wizard 2, you need to run this installer with administrator rights. (Windows users only)
The tool, called FDOToolbox, which is needed to use SDF files will be installed after this installation.
If you do not need this functionality, you may refuse this installation when it starts.
